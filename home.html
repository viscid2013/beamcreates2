Hi! I'm Ben McFadden, and BeAm Creates is my online portfolio and learning labratory for my graphic design and web coding work.

Below you'll find a list of my current and upcoming projects.

Viscid Creates represents a practice project to deepen and apply my knowledge of some raw coding skills.

It is built using PHP with MySQL/PDO, HTML5 with W3-CSS for responsive styling, and mostly Javascript with a touch of Angular and jQuery* where it made things simpler.

*Credit to the originators of the modified code I used is in the comments.
